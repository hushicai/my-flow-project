# workflow

webstorm eslint flow workflow

## eslint

代码规范

```bash
npm install eslint babel-eslint --save-dev
```

### eslint-plugin-flowtype

flow类型标注代码规范

### eslint-plugin-react

react/jsx代码规范

## flow

类型安全系统

### flow-bin

配置project-specified版本flow。

```bash
npm install flow-bin --save-dev
```

### flowconfig

生成`.flowconfig`文件：

```
flow init
```
